# sphere-interactive

A simple interactive sphere demo created using [raylib](https://www.raylib.com/).

## Demo

![Demo](docs/demo.gif)

## Prerequisites

- [raylib](https://www.raylib.com/)
- Any `C99` compliant compiler

## Getting started

```console
$ make release
```

or simply

```console
$ make
```

to build with debug flags.
